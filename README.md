# Green City — Interactive Multi-Step Quiz (Webflow)

Коммерческий интерактивный квиз-лендинг, разработанный для агентства недвижимости **Green City Real Estate** (Дубай, ОАЭ). Проект создан специально для высококонверсионного сбора заявок (лидогенерации) через таргетированную рекламу.

* **Статус проекта:** В продакшене.
* **Ссылка на сайт:** [deals.greencityre.org/ru/](https://greencityre.org)
* **Инструменты разработки:** Webflow, Webflow Interactions.

---

## 🏗 Технические сложности и инженерные решения
Главная фишка этого проекта — реализация пошагового опросника (Multi-step form) с ветвлением логики в условиях жестких ограничений платформы Webflow. 

* **Логика переходов без плагинов:** Так как стандартные формы Webflow не поддерживают пошаговость «из коробки», вся система экранов, переключения между вопросами, кнопки «Далее»/«Назад» и прогресс-бар были реализованы вручную через сложные, каскадные цепочки встроенных анимаций (`Webflow Interactions`).
* **Агрегация и валидация данных:** На каждом шаге квиза данные (цель покупки, бюджет, тип объекта) временно удерживаются в состоянии интерфейса. На финальном шаге происходит валидация контактов пользователя, после чего весь массив собранных ответов упаковывается и отправляется в CRM-систему.
* **Интеграция с CRM и аналитикой:** Настроена сквозная передача данных. В CRM брокерам уходит не просто «голый» телефон, а готовый профиль клиента с его предпочтениями, что кратно увеличивает конверсию отдела продаж. Также подключены пиксели отслеживания для оптимизации рекламы под заполнение квиза.
